\doxysubsection{Detailed Description}
Concrete class handling P1 ASCII \doxylink{classPBM}{PBM} files. 

In \doxylink{classPBM}{PBM} format, images are strictly monochrome (black and white). The specification dictates that \textquotesingle{}0\textquotesingle{} is a white pixel and \textquotesingle{}1\textquotesingle{} is a black pixel.
