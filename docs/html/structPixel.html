Detailed Description

Represents a single RGB Pixel in a raster image.

This structure encapsulates the red, green and blue color channels

  • Architectural note on the data types: We use uint8_t (from <cstdint>), because in the 8 bits that we have given it represents the color channels in a perfect manor (0 - 255) Also we use it because it is MUCH MORE memory efficient, if we were to use int or unsigned this would have cost us 4 bytes (32 bits) we hvae a 75% effieciency increse only by using this variable instead of int
