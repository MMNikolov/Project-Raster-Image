\doxysubsection{Detailed Description}
abstract base class represening a raster image 

This class represents the polymorphic interface for all specific image formats (e.\+g. \doxylink{classPPM}{PPM} -\/ Portable Pixmap Format \texorpdfstring{$\vert$}{|} \doxylink{classPGM}{PGM} -\/ portable Grey map \texorpdfstring{$\vert$}{|} \doxylink{classPBM}{PBM} -\/ Portable Bit\+Map \texorpdfstring{$\vert$}{|} BMP -\/ Bitmap) It encapsulates the core data (width, height and pixels) and defines virtual methods that derived classes must implement Architectural Note:\+ The {\ttfamily \doxylink{classImage_afdcb0becb5c16b16736675825a0f8545}{pixel\+Data}} is stored in a 1D vector rather than a 2D array This boosts performance when applying transformations to large images
